# University Club Analytics Engine (SQL/Python)

## Business Context
Designed a data analytics framework to optimize member engagement for a 500-member university consulting club. 

## Key Insights Generated
* **Talent Sourcing:** Identified Top 5% of "High-Engagement" members for leadership roles.
* **Retention Risk:** Pinpointed 'Economics' and 'Art' majors as the highest churn segments (65% of passive base).

## Tech Stack
* **Database:** SQLite (Relational Design)
* **Analysis:** Python (Custom SQL wrappers)
* **Reporting:** Automated Executive Brief generation
